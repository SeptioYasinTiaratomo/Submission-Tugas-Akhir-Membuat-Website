## ğŸ¯ Kriteria & Fitur Utama

Proyek ini dibangun dengan mematuhi kriteria teknis yang ketat:

### 1. Semantic HTML
Struktur dokumen disusun menggunakan elemen semantik untuk meningkatkan aksesibilitas dan SEO:
- `<header>`: Menampung judul website dan navigasi.
- `<main>`: Membungkus konten utama artikel.
- `<article>`: Digunakan untuk setiap pos artikel dan kartu profil.
- `<aside>`: Menampilkan konten pendamping (profil penulis) dengan gambar.
- `<footer>`: Informasi hak cipta di bagian bawah halaman.

### 2. Layout dengan Flexbox
Tata letak website disusun sepenuhnya menggunakan **CSS Flexbox** (`display: flex`), bukan menggunakan properti `float`. Ini memastikan elemen tertata rapi baik secara horizontal maupun vertikal.

### 3. Responsif (Mobile-Friendly)
Tampilan website dapat beradaptasi dengan berbagai ukuran layar.
- **Desktop:** Layout dua kolom (Main Content di kiri, Aside di kanan).
- **Mobile/Tablet:** Layout otomatis berubah menjadi satu kolom (stack) untuk kenyamanan membaca di layar kecil.
